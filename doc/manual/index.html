Graph::Easy - Manual

Index

Teamwork is the key to > Victory
   

This manual explains the algorithms and ideas behind Graph::Easy, as well as specific details like the syntax rules for the text format.

You will learn in detail how to create a graph by describing it in an easy-to-use and easy-to-understand language, and then convert it to the desired output formats like ASCII, HTML, SVG (Scalable Vector Graphics) or graphviz, which in turn allows you to generate PNG or PDF files.
