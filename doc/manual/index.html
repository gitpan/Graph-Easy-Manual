Graph::Easy - Manual

Teamwork is the key to > Victory
   

You will learn in detail how to create a graph (or flow chart) by describing it in an easy-to-use and easy-to-understand language, and then convert it to the desired output formats like ASCII, HTML, SVG (Scalable Vector Graphics) or graphviz, which in turn allows you to generate PNG or PDF files.

This manual explains the algorithms and ideas behind Graph::Easy, as well as specific details like the syntax rules for the text format.

Demos, Usage and Examples

You can see a live-demo of Graph::Easy. It is also used in projects that graph Perl packages dependencies or usage. Plus, with a Mediawiki-Plugin you can use it inside wikis, or even use it to add graphs to POD.
